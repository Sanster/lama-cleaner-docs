## Windows 1-Click Installer

1. Download [lama-cleaner-win.zip](https://github.com/Sanster/lama-cleaner/releases/download/win_one_click_installer/lama-cleaner-win.zip) and unzip it.

<Callout type="info" emoji="ℹ️">
  Please download this
  [tasks.py](https://raw.githubusercontent.com/Sanster/lama-cleaner/main/scripts/user_scripts/tasks.py)
  and replace the tasks.py file from the zip. The main difference is sd1.5 model
  no more need Hugging Face Access Token
</Callout>

1. Double click `win_config.bat`, it will take a few minutes to install all the dependencies, and ask you to choice model and device.
1. Double click `win_start.bat` to start the server, at first time it take a few minutes to download the model(about 4G).
