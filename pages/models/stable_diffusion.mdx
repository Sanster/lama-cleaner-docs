## Load local ckpt/safetensors

You can learn how to create a inpainting model from **any** diffusion model in AUTO1111's webui [here](https://www.reddit.com/r/StableDiffusion/comments/zyi24j/how_to_turn_any_model_into_an_inpainting_model/).

Then you can load it into Lama Cleaner with following command:

```bash
--model sd1.5 --sd-local-model-path /path/to/model-inpaint.ckpt
```
